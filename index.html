Workflow Guide

1. Creating a Flow: Habits are linked like a chain. Every habit points to the "Next Step" ID.

2. Inserting a Step: Use the "Insert After" button on an existing habit. This automatically creates a new task and rewires the connections so it fits in between.

3. Branching: Change a step type to "Branch". This creates a Yes/No fork. You must specify which habit comes next for Yes and for No.

4. Auto-Organize: Click "Auto-Organize" to sort the list below visually to match the actual flow order.

5. Multi-Assign: Select multiple people to create tracking lanes for everyone.
