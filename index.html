Workflow Guide

1. Creating a Flow: Habits are linked like a chain. Every habit points to the "Next Step" ID.

2. Reordering: Drag and drop steps in the editor or use the arrow buttons. The list is just for editing; the actual execution flow is determined by the "Next Step" logic links.

3. Branching: Change a step type to "Branch". This creates a Yes/No fork.

4. Assignments: Assign habits to specific household members. You can pause specific people or set individual targets.

5. Importance: Mark key habits with the star icon to highlight them in yellow.
